Чтобы развернуть проект на «чистом» компьютере в аудитории, тебе понадобится установить следующий набор программ и выполнить несколько шагов для запуска.

💻 1. Что нужно установить
Node.js (желательно LTS версию) — необходим для работы и бэкенда, и фронтенда, и содержит пакетный менеджер npm.
PostgreSQL — система управления базами данных (СУБД).
Редактор кода (например, VS Code, Cursor или WebStorm) — для работы с кодом проекта.
Браузер (Google Chrome, Firefox и т.д.) — для проверки работы фронтенда.
🚀 2. Как подготовить и запустить проект
Шаг 1: Настройка базы данных (PostgreSQL)
Судя по файлу backend/.env, проект ожидает конкретные доступы к базе данных. При установке PostgreSQL:

Оставь пользователя по умолчанию: postgres
Задай пароль: 010406 (обязательно такой, иначе придется менять .env)
Порт оставь стандартный: 5432
После установки открой pgAdmin (или консоль SQL Shell) и создай пустую базу данных с именем exam.
Скрипт бэкенда при запуске сам создаст все нужные таблицы (users, bookings, reviews) и администратора (логин Admin26, пароль Demo20), так что вручную таблицы создавать не нужно.

Шаг 2: Запуск бэкенда
Открой проект в редакторе кода.
Открой терминал и перейди в папку бэкенда:
cd exam/backend
Установи зависимости:
npm install
Запусти сервер:
npm run dev
Должно появиться сообщение: "Сервер запущен на порту 5000" и "База данных инициализирована".
Шаг 3: Запуск фронтенда
Открой новый (второй) терминал в редакторе кода и перейди в папку фронтенда:
cd exam/frontend
Установи зависимости:
npm install
Запусти проект:

npm run dev
